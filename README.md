# bingExcel  

处理excel与java model之间转换,想做成一个excel的orm类型的框架  
具体使用说明可以参照 [wiki](https://github.com/bingyulei007/bingExcel/wiki)
## 环境说明
基于java1.7，依赖poi3.1X.jar包,commons-lang3 jar,com.google.guava 等
